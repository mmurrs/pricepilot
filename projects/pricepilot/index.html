Save more on every buy

Find the cheapest Amazon listing — verified.

A pay-per-call price agent for shopping bots. Name a product, PricePilot resolves the exact variant on Amazon and returns the cheapest verified offer — fresh price, real stock, real seller. Walmart and Target coming next.

$0.05 /check · paid in USDC via x402 or MPP

What your agent can ask today

Anything Amazon sells with a clear brand + model + variant. The more specific your spec, the cleaner the match.

Cheapest Nike Killshot 2 Sail/Lucid Green men's size 11.5
Cheapest Sony WH-1000XM5 headphones in black
Cheapest Hoka Clifton 9 women's size 8
Cheapest LEGO 10497 Galaxy Explorer set

The endpoint

Cheapest Product
$0.05 / check
POST/find_cheapest

Pass an explicit product spec — brand, model, variant attributes (color, size, storage). Today PricePilot resolves the exact variant on Amazon (SERP → parent PDP → child PDP) and returns the verified buyable offer. More retailers fan out from the same call as we add them.

Amazon — live Walmart — next Target — planned

Every response includes a stable product_id so your agent can store, compare, or watch a product over time.

What happens on a /find_cheapest call

01
You send a product spec. Brand, model, variant fields (size, color, storage). No URLs to guess, no SKUs to look up.
02
We resolve the exact variant on Amazon. SERP → parent PDP → child PDP, so the offer matches your spec — not a near-miss.
03
We pull the live offer. Real price, real stock, real seller from the PDP at request time. Not a cached SERP estimate.
04
You get one JSON response. The cheapest verified offer plus a stable product_id you can reuse.

Make your first call

1

Pick a wallet your agent can pay from

AgentCash is the fastest — it ships USDC and signs payments for any agent. Or use your own Base / Tempo wallet directly via x402 / MPP.

2

POST a product spec to /find_cheapest

The first request gets a 402 with the price ($0.05). Re-send with the payment header and you get the cheapest offer.

$ curl -X POST https://pricepilot.vercel.app/find_cheapest \ -H "Content-Type: application/json" \ -d '{"brand":"Nike","model":"Killshot 2","size":{"system":"US","gender":"men","value":11.5}}'
3

Read the cheapest offer from the response

The best field is the verified buyable offer. Hand it to your shopper, log it, or check again tomorrow.

How to pay

1

Onboard once

Creates a wallet, funds it with test USDC, signs payments for you.

$ npx agentcash onboard
2

Try a paid call

Runs the curl above and pays the 402 automatically.

$ npx agentcash try https://pricepilot.vercel.app/find_cheapest
1

Have USDC on Base mainnet (or Sepolia)

Any EOA or smart wallet with a private key your agent can sign with.

2

Wrap fetch with x402-fetch

import { withPaymentInterceptor } from 'x402-fetch' const fetchWithPay = withPaymentInterceptor(fetch, account) await fetchWithPay('https://pricepilot.vercel.app/find_cheapest', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(spec), })
1

Have USDC on Tempo

Any Tempo wallet your agent can sign from. Privy server wallets work.

2

Configure mppx and call

import { Mppx, tempo } from 'mppx/client' Mppx.create({ methods: [tempo({ account: tempoWallet })] }) await fetch('https://pricepilot.vercel.app/find_cheapest', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(spec), })
